# Algonive_Customer_Segmentation (Task-1)
Customer segmentation using K-Means clustering

## 📌 Project Overview
This project focuses on segmenting customers into different groups based on their behavior and purchasing patterns. The goal is to identify distinct customer segments to help businesses target the right audience effectively.

---

## 🎯 Objective
- Group customers based on similarities  
- Identify different customer segments  
- Understand customer behavior  
- Help in personalized marketing strategies  

---

## 📂 Dataset
- Customer dataset containing:
  - Customer ID  
  - Age / Gender (if applicable)  
  - Annual Income  
  - Spending Score  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Jupyter Notebook  

---

## 🔍 Steps Performed

### 1. Data Preprocessing
- Checked for missing values  
- Selected relevant features  
- Scaled the data (if applied)  

### 2. Finding Optimal Clusters
- Used **Elbow Method** to determine optimal number of clusters  

### 3. Model Building
- Applied **K-Means Clustering Algorithm**  
- Assigned customers to clusters  

### 4. Visualization
- Plotted clusters using scatter plots  
- Visualized customer segments  

---

## 📊 Key Insights
- Customers are grouped into distinct segments based on income and spending  
- Identified high-value customers  
- Found low-spending and high-spending groups  

---

## 📈 Conclusion
Customer segmentation helps businesses understand their customers better and create targeted marketing strategies. This project demonstrates how machine learning can be used to segment customers effectively.
